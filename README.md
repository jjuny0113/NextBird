# Nextweet

### 기간 : 2020년 2월 15일 ~ 2021년 3월 19일

### 내용 : SNS서비스 'twitter'를 모티브로 한 프로젝트

### 기술스택
  - Next.js
  - Redux, Redux Saga
  - Styled-components
  - Ant Design
  
### 구현내용

- 서버사이드 렌더링 
- custom hook을 이용해 input관리
- 게시글 작성,삭제
- 게시물 댓글 작성, 삭제
- 해시태그 링크화(정규표현식 사용)
- 인피니티 스크롤 적용
- Formdata를 이용해 이미지 업로드
- 회원가입, 로그인, 로그아웃 기능
- 팔로우, 언팔로우 구현
- 리트윗
